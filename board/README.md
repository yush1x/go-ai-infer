# Board 类
### move 落子

处理落子，还要提子，坐标是0-18+pass

如果不合法，抛出异常

### mask 查看合法点

返回一个0-1矩阵，0代表不能下，19*19长度的向量

### tensor 转换为可以直接传递给推理服务的类型

返回值参考推理服务的要求

返回inference.Features，它里面本质是[3249]float32

### score 数目数

根据联通块，周围全是同色就归谁，不然中立

返回三个数，黑/白/中立的个数

### is_finish 判断终局

最后两个是pass，就终局

### is_eyes 判断是不是眼

把对方棋子下不进去就叫眼
